# ASRLang
My little language, as I wanted to learn how languages crafted, ( tree-based, and bytecode (WIP) ) versions implemented ^_^
implemented a custom dynamically-typed programming language. featuring a tree-walk interpreter in Java and a performance-oriented bytecode VM in C, covering full pipeline design from lexical analysis and AST construction to execution and virtual machine implementation
## source
inspired by the great book Crafting Interpreters by Robert Nystrom check the [book](https://www.craftinginterpreters.com)
